容联云通讯短信平台

特别注意:

想要使用容联云发送短信, 需要满足以下条件:

  1. 在容联云注册账号.

  2. 登陆后, 在容联云官网创建一个子应用, 并且需要提交子应用

使其成为运营中状态, 否则不成功

  1. 把自己的手机号添加到官网的测试号码中

容联云配置

我们想要使用容联云, 需要把老师发的资料中

的 yuntongxun 文件夹拷贝到项目中 libs 包中

其中, yuntongxun有两个重要文件:

CCPRestSDK.py ：

由容联云通讯开发者编写的官方 SDK 文件，包括发送模板短信的方法

补充: SDK是软件开发工具包的意思, 主要指别人写好, 供我们调用的文件.

ccp_sms.py ：

发送模板短信的方法封装在这个文件中. 我们要用的方法, 也在这个文件中.

容联云测试:

我们可以在 ccp_sms 文件的最下面, 找到 main, 如下: 

if __name__ == '__main__':
    # 注意： 测试的短信模板编号为1
    CCP().send_template_sms('在官网添加的测试号码', ['验证码', 5], 1)

说明:

测试号码, 一定要在官网添加过, 否则不成功. 我们这是测试状态, 免费的.

所以只能给测试号码发送短信. 其他的不行. 可以设置三个测试号码.

验证码可以随意写一个, 也可以随机生成, 4位或6位都可以, 中文也可以.

这里的 5 指的是有效期

这里的 1 指的是我们使用的模板编号. 免费状态, 模板编号都是1.

想要知道模板是啥, 可以查看官网.

总结:

  1. 容联云通讯只是发送短信的平台之一，还有其他云平台可用，比如，阿里云等，实现套路都是相通的。
  2. 将发短信的类封装为单例，属于性能优化的一种方案。
